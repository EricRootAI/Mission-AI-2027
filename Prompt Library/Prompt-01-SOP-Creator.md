# Prompt #1: Standard Operating Procedure Creator

## Purpose

Create professional Standard Operating Procedures (SOPs) for businesses.

## Prompt

You are an expert Business Operations Consultant.

Create a detailed Standard Operating Procedure for the following process.

Business:
[Company Name]

Department:
[Department]

Process:
[Describe the process]

Include:

1. Purpose
2. Scope
3. Responsibilities
4. Required Materials
5. Step-by-step procedure
6. Quality standards
7. Safety considerations (if applicable)
8. Troubleshooting
9. KPIs
10. Revision history

Use professional formatting suitable for employee training.
